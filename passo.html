Guia para Montagem do Projeto

Esse projeto envolve o uso de LEDs, botões, resistores, um buzzer e uma protoboard, com o objetivo de criar um sistema onde apenas um botão pode ser ativado por vez, sob controle de um mediador. Durante a ativação, um contador de 13 segundos é iniciado, e um aviso sonoro é emitido para indicar o término do tempo.

Objetivo do Projeto

Criar um sistema interativo com Arduino onde:

  • Cada pessoa possui um botão associado.
  • Apenas um botão pode ser pressionado por vez.
  • O mediador controla quando outro participante pode pressionar seu botão.
  • Um buzzer emite sons intermitentes durante 13 segundos para indicar o tempo disponível.
  • No final do tempo, o buzzer emite um som longo para avisar que o tempo de resposta se encerrou.

Materiais necessários

  • 1 Arduino Uno (ou similar).
  • 1 Protoboard.
  • Botões push-button (1 por jogador, 1 para o mediador).
  • LEDs (1 para cada jogador).
  • 1 Buzzer piezoelétrico.
  • Resistores de 220Ω.
  • Jumpers (fios de conexão).
  • Cabo USB (para conectar o Arduino ao computador).

Passo 1

Testar o Circuito no Tinkercad Antes de montar fisicamente, recomendamos usar o Tinkercad para simular o funcionamento do circuito:

  1. Acesse o TinkerCad
  2. Crie um novo projeto e adicione os componentes mencionados.
  3. Configure o circuito para:
    • Cada botão acender um LED específico.
    • Impedir a ativação simultânea de LEDs.
  4. Faça ajustes no código do Arduino no ambiente de simulação

Passo 2

Conectar o Arduino na Protoboard

  • Faça as conexões básicas de alimentação:
    • 5V do Arduino para a linha positiva da protoboard.
    • GND do Arduino para a linha negativa.
  • Imagem: Montagem inicial (protoboard e Arduino).

Passo 3

Conectar os LEDs

  • Insira os LEDs na protoboard (um por jogador).
  • Conecte o ânodo de cada LED a um resistor de 220Ω, ligado a um pino digital do Arduino (ex.: pinos 2, 3, 4…).
  • Conecte o cátodo de cada LED à linha negativa (GND).

Passo 4

Conectar os Botões

  • Conecte cada botão à protoboard.
  • Ligue um lado do botão a um pino digital do Arduino (ex.: pinos 5, 6, 7…).
  • Conecte o outro lado à linha positiva (+5V).
  • Adicione um resistor de 220kΩ entre o pino do botão e o GND.

Passo 5

Ajustes Finais

  • Organize os fios e componentes na protoboard para evitar curto-circuitos.
  • (Opcional) Monte o circuito em uma placa PCB para uma versão mais permanente.

Passo 6

Conectar o Buzzer

  • Escolha o pino do Arduino:
    • Utilize o pino 9 como saída digital para o buzzer.
  • Conexões do Buzzer:
    • Conecte o terminal positivo do buzzer ao pino 9 do Arduino.
    • Conecte o terminal negativo ao GND do Arduino.
    • Se o buzzer for passivo, adicione um resistor de 220Ω em série.

Passo 8

Testar o Circuito

  • Faça o upload do código para o Arduino.
  • Pressione um botão para iniciar a contagem de 13 segundos.
  • Observe os bipes do buzzer a cada segundo e o som longo no final do tempo.
  • Teste o botão do mediador para resetar o sistema antes do fim da contagem.

Resultados Esperados

  • Um LED acende e o buzzer emite bipes curtos intermitentes a cada segundo enquanto o tempo passa.
  • Após 13 segundos:
    • O buzzer emite um som longo para indicar que o tempo acabou.
  • O sistema permite reset manual pelo mediador a qualquer momento.
